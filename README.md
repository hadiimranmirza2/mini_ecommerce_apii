# Mini E-Commerce API
[cite_start]An engine behind an online store handling products, categories, and orders[cite: 7].

## Setup Instructions
1. Install dependencies: `pip install -r requirements.txt`
2. Run migrations: `python manage.py migrate`
3. Start server: `python manage.py runserver`

## Documentation
[cite_start]Access the Swagger UI at `http://127.0.0.1:8000/api/docs/`[cite: 55].
